Registry Tiny Theme
===================

Files included:
- style.css
- functions.php
- header.php
- footer.php
- front-page.php
- page.php
- index.php
- template-parts/components/hero.php

Install:
1. Upload the folder to wp-content/themes/ or upload the zip in Appearance > Themes.
2. Activate the theme.
3. Set a static homepage in Settings > Reading if you want front-page.php to drive the home page.
4. Create and assign a menu to the Primary Menu location if desired.

This is intentionally minimal. Add templates and component files as the site grows.
